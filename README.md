## Setup

### First time

```bash
$> npm run setup
```

### Dependencies

```bash
$> npm install
```

## .NET

### Configurations

* `Debug`
* `Release`

### Build

```bash
$> npm run dotnet:build:(Debug|Release)
```

### Run

Open 2 terminals

1. `$> npm start`
1. `$> npm browser`

### Publish

```bash
$> npm run dotnet:publish
```
